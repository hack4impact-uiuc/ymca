# YMCA API

This folder contains the backend api of the application.

## Install & Run

To set up, first `cd` into this directory. Then,

```bash
yarn
yarn start
```

This will create a server on [http://localhost:9000](http://localhost:9000).

## Technologies

Built using Express and MongoDB.
